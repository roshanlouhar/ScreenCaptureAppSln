1. Install the setup on system.
2. Go to the directory "C:\Program Files (x86)\Increpe\TopoScreenInstaller\" folder where all files available.
3. Install the Mysql Server on user machine Manually.
4. Run the Mysql command line argument.
5. Run the source file localDatabseScript file from installer directory.
6. Execute the CloudDatabseScript file on Server.
7. Now open the "ScreenCaptureApp.exe" xml file and edit the connection strings.
8. Run the exe.
